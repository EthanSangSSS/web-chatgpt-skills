# Category Matrix

Use this file as a quick reference for product-specific evaluation dimensions. Do not paste the whole matrix into final answers; select only dimensions that affect the user's decision.

## Universal Dimensions

| Dimension | What to verify |
|---|---|
| Fit | Primary use, constraints, ecosystem, installation or compatibility. |
| Value | Current price, historical low, same-price alternatives, previous-gen alternatives. |
| Reliability | Common failures, return rates if available, warranty, repairability, owner reports. |
| Ownership cost | Accessories, consumables, subscriptions, repair, resale value. |
| Privacy/software | Ads, telemetry, app permissions, update record, cloud dependency. |
| Timing | Launch maturity, replacement cycle, sale season, return window. |

## Phones

| Dimension | Checks |
|---|---|
| Performance | Sustained performance, thermals, real games/apps, 1% low, battery drain. |
| Camera | Day/night samples, shutter lag, video stability, processing artifacts. |
| Battery/charging | Real screen-on time, heat while charging, charger included, battery health behavior. |
| Software | Update policy, ads, background killing, ecosystem lock-in. |
| Region | Bands, NFC, eSIM, warranty, preinstalled software. |

## Laptops

| Dimension | Checks |
|---|---|
| Workload fit | CPU/GPU sustained power, RAM/storage ceiling, fan noise, thermal throttling. |
| Screen | Brightness, color, PWM/flicker, refresh, aspect ratio. |
| Keyboard/trackpad | Layout, travel, reliability, palm rejection. |
| Ports | Dock needs, charging standard, external monitor support. |
| Repair/resale | Battery replacement, SSD/RAM upgrade, warranty, second-hand value. |

## PC Components And AI Hardware

| Dimension | Checks |
|---|---|
| Workload bottleneck | Gaming resolution/refresh, VRAM need, model size, context length, batch size, render/export workload. |
| Platform cost | PSU, connector standard, case clearance, cooling, motherboard/CPU bottleneck, RAM, SSD. |
| Market regime | MSRP vs street price, stock depth, launch timing, GDDR/DRAM/NAND pressure, tariffs/export controls. |
| Local vs cloud | Expected hours, privacy, latency, cloud hourly rate, setup overhead, break-even point. |
| Software ecosystem | CUDA/ROCm, driver stability, framework support, firmware/VBIOS issues. |
| Resale/liquidity | Warranty transfer, used market demand, depreciation, wear stigma. |

## TVs And Monitors

| Dimension | Checks |
|---|---|
| Panel quality | Real brightness, contrast, local dimming zones, uniformity, viewing angle. |
| Motion/gaming | Input lag, VRR, 1% lows for gaming displays, response time, HDMI bandwidth. |
| HDR | Peak vs sustained brightness, tone mapping, blooming, HDR format support. |
| Smart system | Ads, startup speed, app support, privacy, update record. |
| Fit | Viewing distance, mounting, room brightness, console/PC compatibility. |

## Headphones And Speakers

| Dimension | Checks |
|---|---|
| Sound | Frequency response, codec support, tuning preference, distortion. |
| Comfort | Weight, clamp, heat, glasses compatibility, long-session fatigue. |
| Noise control | ANC strength, transparency quality, wind noise, mic quality. |
| Reliability | Battery degradation, hinge/cable failures, replacement earpads. |
| Ecosystem | Multipoint, latency, platform-exclusive features. |

## Routers And Smart Home

| Dimension | Checks |
|---|---|
| Coverage | Home layout, wall material, mesh need, wired backhaul. |
| Stability | Firmware history, reboot complaints, device count, heat. |
| Security | Update cadence, admin exposure, cloud dependency, privacy. |
| Compatibility | ISP mode, VLAN/IPTV, Matter/HomeKit/Alexa/Google support. |
| Manageability | App quality, local web UI, parental controls, logs. |

## Appliances

| Dimension | Checks |
|---|---|
| Core function | Cleaning/cooling/heating/washing performance in real households. |
| Noise and installation | Decibels, space, drainage, ventilation, maintenance access. |
| Consumables | Filters, bags, detergent, proprietary parts, subscription risk. |
| Reliability | Compressor/motor/pump failures, warranty, repair network. |
| Smart features | Whether app/cloud features are necessary or fragile. |
