# Knowledge Comic
## Workflow
1. Define learning objective, reader, key misconception, and desired emotional payoff.
2. Reduce the topic to a narrative arc: hook, tension, explanation, example, resolution.
3. Create a panel-by-panel storyboard with dialogue, narration, visual action, and transition.
4. Create consistent image prompts for the characters, environment, palette, and panel composition.
## Output
- Premise and learning objective
- Character and visual bible
- Panel storyboard
- Image prompts and accessible alt text
## Rules
Each panel must earn its place. Keep text short, explain one concept at a time, and never claim historical facts without a source.
