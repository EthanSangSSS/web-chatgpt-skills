# Article Illustrator
## Workflow
1. Read the article and identify its thesis, emotional arc, audience, and 3-6 illustration moments.
2. Choose one visual system: style, palette, composition, recurring motifs, and accessibility approach.
3. For each image, define narrative purpose, placement, subject, composition, text constraints, and alt text.
4. Produce an image-generation prompt that preserves the shared visual system.
## Output
- Visual direction
- Illustration map table
- One production-ready prompt per illustration
- Alt text and caption guidance
## Rules
Illustrations must clarify or pace the article, not decorate every section. Avoid unreadable text inside generated images and avoid copying living artists' styles.
