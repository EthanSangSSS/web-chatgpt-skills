# Infographic Designer
## Workflow
1. Define the single question the graphic answers and the intended audience.
2. Audit data: source, date, unit, denominator, uncertainty, and whether comparisons are valid.
3. Select a layout: comparison, timeline, process, hierarchy, map, scorecard, or dashboard.
4. Establish visual hierarchy: title, one takeaway, primary data, supporting detail, source note.
5. Produce a visual specification and image-generation prompt.
## Rules
Do not use decorative charts. Label units and sources, avoid misleading scale choices, and state uncertainty when it changes interpretation.
## Output
A wireframe, data table, style guide, production prompt, and alt text.
